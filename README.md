# AI Email Reply Agent

This is a simple Python AI agent that automatically generates professional email replies.

## Features

- Generate replies using AI
- Simple Python script
- Easy to modify for automation

## Installation

Clone the repository

git clone https://github.com/yourusername/ai-email-agent.git

Install dependencies

pip install -r requirements.txt

## Usage

Add your OpenAI API key inside the script.

Run the program

python email_agent.py

The AI will generate a reply for the email.

## Example Output

Hi John,

Thanks for your message. Tomorrow works for me.  
Please let me know a convenient time.

Best regards
